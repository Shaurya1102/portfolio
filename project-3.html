Project Overview

A fitness website, made using HTML, CSS, JavaScript, Node.js, Express.js, MongoDB and Vite.js.

This website has many features. It has an inbuilt BMI calculator, and the previous BMI are stored in the Database present in MongoDB. There is a full fledged login and registration fucntionality, which is also connected to MongoDB. There is a latest AI plan feature, made by using OpenAI API, used to generate the plans of a user with keeping their needs in mind to make the diet plan generated as close to their needs as possible.
